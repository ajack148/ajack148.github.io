Project Name: Author Website

Project Overview

This application is a website curated for my client, an aspiring author. The website will promote her upcoming book and give users some intel about who she is and where she came from. The intended users of this website will be book readers looking for a new author to explore.

The website will include five pages: a Home page, a Bio page, a Contact page, a FAQ page, and a Works/Upcoming Books page.

Page Design

Home Page – Sarah Parish

This page introduces Sarah Parish and her upcoming book. It includes a newsletter sign-up form with input validations, including radio buttons and form submission feedback for user engagement.

Books / Works - Works by Sarah Parish

This page showcases Sarah's upcoming and current works. It informs visitors about her releases and may include a pre-order button in the future that leads to a cart or confirmation page.

About Me

This page shares Sarah Parish’s background, writing journey, and interests. It is static and meant for readers who want to know more about the author.

FAQ

This page answers frequently asked questions from readers. It serves as a resource but has no interactive elements or user input fields.

Contact Me

This page lists ways to contact Sarah Parish, such as email or social media links. It includes hyperlinks but does not require input or validation.

Dynamic Functionality Using JavaScript

JavaScript will enhance interactivity on the website, especially on the Home Page. A newsletter form will be implemented to collect user details like name, email, and frequency of updates. Validation will ensure required fields are completed and emails are valid.

Another potential feature is a pre-order button on the Works page, which may dynamically redirect users to a cart page or show a pop-up confirmation using JavaScript.
