Lucas Rettke's Review: By Aaliyah Jackson

Rettke, Lucas

Review Points for Website

  • The link I was provided led straight to their ITIS 3135 homepage with all their links. I took a look at all their links and found the Client Project link which led me straight to their website.
  • No obvious spaces or uppercase letters in URLs or file names.
  • Good design:
    • Sufficient contrast and font sizing, the text was very easy to read.
    • Consistent color scheme and fonts. I checked their GitHub and saw they used a separate CSS file for their styles.
    • All CRAP principles followed (Contrast, Repetition, Alignment, Proximity).
  • Header, main content, and footer are present!
  • Navigation menu was in between header and main content visually and within the code.
  • Header contains the site/brand name correctly.
  • This was not a homework page, so they didn't need to list student information.
  • Main content starts with a heading(h2), Introducing the user to VOICE. The student understood the difference between head, header, and heading.
  • Brand tagline was not clearly visible? I do see some taglines but they are not consistent on every page.
  • Footer exists but does not include a menu for user pages, only the branding link (the whimsical company), which is acceptable here.
  • Has the HTML validation link.
  • Has the CSS validation link.
  • Overall structure, style, and organization are very strong! It is very easy to navigate!
  • There are many different pages with working navigation links.
  • The navigation bar is very noticeable and easy to use.
  • Plenty of images bring more life and energy to the pages.
  • Have the interactive elements that were needed for this project. They have a form that users can use to submit information, and a slideshow! However, I think the project assignment wants us to have 3 interactive elements, so they should add one more for safe measures!
  • They have plenty of interactive links.
