Nik Turner's Review: By Aaliyah Jackson

Turner, Nik

Review Points for Website

  • The link led straight to their ITIS 3135 homepage with all their links! I was able to find the webpage easily that way.
  • No obvious spaces or uppercase letters in URLs or file names.
  • Good design:
    • Sufficient contrast and font sizing, the text was very easy to read.
    • Consistent color scheme and fonts. I checked their GitHub and saw they used a separate CSS file for their styles.
    • All CRAP principles followed (Contrast, Repetition, Alignment, Proximity).
  • Header, main content, and footer are present!
  • Navigation menu was not strictly between header and main content. It seems like it was visually within the header, but in the code, it was inside the main section.
  • Header contains the site/brand name correctly.
  • This was not a homework page, so they didn't need to list student information.
  • Main content starts with a heading (h1) and subheading (h2). The student understood the difference between head, header, and heading.
  • Brand tagline was not clearly visible. If there was one, it was not consistent across every page.
  • Footer exists but does not include a menu for user pages, only the branding, which is acceptable here.
  • Missing HTML validation link in some pages.
  • Missing CSS validation link in some pages.
  • Overall structure, style, and organization are very strong!
  • There are many different pages with working navigation links.
  • The navigation bar is very noticeable and easy to use.
  • Plenty of images bring more life and energy to the pages.
  • Some images do not load (for example, in the "leathersmithing" project section).
  • They have interactive elements which I thought was very creative and cool! There was a timeline area where you can click on each year to find out what the client did that specific year. There was also a slider on one of the project pages, however I wasn't sure on what it did. And the contact page had a form where the user could send the client a message.
